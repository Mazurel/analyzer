# Log analyzer

This is a repo for our reasearch project 5@KAMS.

![Homepage](./imgs/homepage.png)

## Local installation

```shell
python -m venv venv
pip install -e .
pip install -e ".[dev]"
python main.py
```

## Running tests

This project uses `pytest` for testing.
To run full test suite, run:

```shell
python test.py
```
