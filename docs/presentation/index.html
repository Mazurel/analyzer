Quickly: What are logs ?

Each application produces text files, line by line, that represent current state of a program.

These lines are usually corresponding to one line inside source code.

Based of them, among the others, we can reason about the program !

Quickly: What are logs ?

For example, a line can clook like this:

                    
                        17.12.2023 14:13 : INFO : Started special service

Lines usually contain timestamp, additional metadata and line content

The biggest issue - there is no common format
